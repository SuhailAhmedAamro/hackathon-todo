# UI Specifications - Claude Guide

## Purpose
UI specs define user interface components, pages, and design system.

## Files in This Directory
- `components.md` - Reusable UI component library
- `pages.md` - Page layouts and routing
- `chat-interface.md` - Chatbot UI design

## How to Use
1. Read UI spec before creating components
2. Follow design tokens and styling
3. Implement accessibility requirements
4. Test responsive behavior
5. Update spec when design changes

## Creating a New UI Spec
Use the template: `@.spec-kit/prompts/ui-spec-template.md`
