# Feature Specifications - Claude Guide

## Purpose
Feature specs define what the application does from a user perspective.

## Files in This Directory
- `task-crud.md` - Create, read, update, delete tasks
- `authentication.md` - User registration and login
- `chatbot.md` - Conversational task management
- `priorities-tags.md` - Task organization
- `recurring-tasks.md` - Recurring task scheduling

## How to Use
1. Read the relevant feature spec before implementing
2. Verify acceptance criteria
3. Check phase applicability
4. Reference related API/database/UI specs
5. Update spec if implementation diverges

## Creating a New Feature Spec
Use the template: `@.spec-kit/prompts/feature-spec-template.md`
