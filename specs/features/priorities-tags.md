# Feature: Priorities and Tags

## Overview
Organize tasks using priorities and custom tags.

## Priorities
- Low, Medium, High
- Visual indicators (colors, icons)
- Filter and sort by priority

## Tags
- User-defined tags
- Color-coded
- Multiple tags per task
- Filter by tags
- Tag management (CRUD)

## User Stories
- As a user, I want to set task priorities so I know what's most important
- As a user, I want to create custom tags to organize my tasks
- As a user, I want to filter tasks by tag and priority
